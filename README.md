# Telecom Customer Churn & Retention Analysis (Power BI)

## Project Overview

This project analyzes telecom customer data using Power BI to understand customer churn behavior and identify key factors influencing customer retention.
The goal is to transform raw customer data into meaningful insights that help businesses reduce churn and improve customer loyalty.


## Objectives

- Analyze customer churn patterns
- Identify factors influencing customer retention
- Evaluate contract types, payment methods, and pricing
- Understand customer lifetime (tenure) behavior
- Provide actionable business recommendations


## Dataset Description

The dataset contains telecom customer information including:

| Column | Description |
|------|------|
| CustomerID | Unique identifier for each customer |
| Gender | Customer gender |
| SeniorCitizen | Indicates if customer is a senior citizen |
| Tenure | Number of months the customer has stayed |
| MonthlyCharges | Monthly subscription cost |
| TotalCharges | Total amount paid by the customer |
| Contract | Type of contract |
| PaymentMethod | Method of payment |
| InternetService | Type of internet service |
| Churn | Indicates whether customer left (Yes/No) |


## Tools Used

- Power BI – Data visualization and dashboard creation  
- Power Query – Data cleaning and transformation  
- DAX – Creating measures and calculated columns  

## Data Preparation
The dataset was cleaned and prepared using the following steps:

- Removed duplicate records  
- Handled missing values  
- Corrected data types  
- Created DAX measures for KPIs  

## Key Measures (DAX)

- Total Customers  
- Churned Customers  
- Retained Customers  
- Churn Rate (%)  
- Total Revenue  
- Average Monthly Charges  

## Dashboard Visualizations

The Power BI dashboard includes:

- KPI Cards (Total Customers, Churn Rate, Revenue)  
- Customer Churn Distribution  
- Churn by Contract Type  
- Churn by Payment Method    
- Monthly Charges vs Tenure  
- Customer Tenure Distribution  

## Key Insights

- High churn rate indicates a retention challenge  
- Month-to-month contracts have the highest churn  
- Customers with higher monthly charges are more likely to churn  
- New customers (low tenure) churn more frequently  
- Long-term customers show stronger loyalty  
- Certain payment methods are linked to higher churn   
- Churn is concentrated in specific customer segments  

## Business Recommendations

- Encourage customers to switch to long-term contracts  
- Improve onboarding for new customers  
- Offer retention incentives and loyalty programs  
- Review pricing strategies for high-cost plans  
- Improve billing and payment experience
